# Elzero Template (Landing Page)

Responsive multi-section landing page built with HTML and CSS (Elzero template).

## Features
- Clean layout with multiple sections (Services / Portfolio / About / Pricing / Contact)
- Fully responsive design
- Font Awesome icons + Normalize.css
- Simple and clean UI

## Tech Stack
- HTML5
- CSS3
- Normalize.css
- Font Awesome
